# octowiz-v2

Octowiz-v2 is the monorepo for the multiplayer AI engineering system — the coding tentacle of ÆLLI. See [`PROJECT_PLAN.md`](PROJECT_PLAN.md) and the canonical docs under [`docs/`](docs/) for product direction, architecture, and MVP scope.

## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues in `raelli/octowiz-v2`, managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the canonical triage vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
